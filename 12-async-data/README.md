## 完整示例代码

```swift
import SwiftUI

struct Chapter12CaseView: View {
    @State private var input = ""
    @State private var items: [String] = []

    var body: some View {
        VStack(spacing: 10) {
            TextField("输入内容", text: $input)
                .textFieldStyle(.roundedBorder)
            Button("添加") {
                guard !input.isEmpty else { return }
                items.append(input)
                input = ""
            }
            List(items, id: \.self) { Text($0) }
        }
        .padding()
    }
}
```
## 代码拆解（小白重点）

- 通过 @State 保存会变化的数据。
- 交互发生后先改状态，再让界面自动刷新。
- 页面结构优先保证清晰，再逐步加样式。
