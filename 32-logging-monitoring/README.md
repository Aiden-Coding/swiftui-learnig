## 完整示例代码

```swift
import SwiftUI

struct Chapter32CaseView: View {
    @State private var enabled = false

    var body: some View {
        Form {
            Toggle("启用本章功能", isOn: $enabled)
            if enabled {
                Text("日志与监控：可观测性入门 已启用")
            } else {
                Text("日志与监控：可观测性入门 未启用")
                    .foregroundStyle(.secondary)
            }
        }
    }
}
```
## 代码拆解（小白重点）

- 通过 @State 保存会变化的数据。
- 交互发生后先改状态，再让界面自动刷新。
- 页面结构优先保证清晰，再逐步加样式。
