## 完整示例代码

```swift
import SwiftUI

struct TopicNavigationView: View {
    let topics = ["布局", "状态", "导航"]

    var body: some View {
        NavigationStack {
            List(topics, id: \.self) { topic in
                NavigationLink(topic) {
                    Text("你选择了：\(topic)")
                        .navigationTitle(topic)
                }
            }
            .navigationTitle("SwiftUI 主题")
        }
    }
}```

## 代码拆解（小白重点）

- NavigationStack 提供页面层级。
- NavigationLink 点击后进入详情页。
- navigationTitle 让层级更清楚。
